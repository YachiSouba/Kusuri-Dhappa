This is the 2D concept of Kusuri Dhappa;

<img width="833" height="562" alt="2d concept art" src="https://github.com/user-attachments/assets/0e650764-5d73-4f0f-b9eb-20d4ac42afbb" />  

So the OLED screen here will display time normally like a clock, with expressive faces.  
When it starts ringing, it will display subtitles for her speech with really cute illustrations.  
The button here helps in turning off the alarm and to help the set the time.  
The rotary encoder is used for setting up time.   
I hope we all know the usage of the speaker but to be precise it is 36mm one.  
While the USB port helps in recharging the battery inside.  
The sides of Kusuri Dhappa have no functional buttons.  

What exactly is the reason behind the name?????  
Well, "Kusuri" means medicine in Japanese.    
while, "Dhappa" is a Hindi word which showcases a loud sound or a line said by the chaser when playing tag (pakdum pakdai) when they catches the player.
